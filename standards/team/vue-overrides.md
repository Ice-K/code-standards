# Vue 团队定制覆盖

在此文件中添加团队对 Vue 规范的定制规则。此文件优先级高于 `standards/vue3/` 和 `standards/vue2/` 下的默认规范。

## 使用方式

1. 复制下方的规则模板
2. 填写团队特有的规则
3. 此文件由团队维护，插件升级不会覆盖

## 规则模板

```
### R{N}: {规则标题}

- **级别**: 必须 | 推荐 | 建议
- **覆盖**: 覆盖默认规范中的 R-XXX-XX 规则（如适用）
- **描述**: {规则说明}

**正确示例：**
{代码}

**错误示例：**
{代码}
```
